# Module Specifications

## 1. 7-Color LED Flash Module
- Operating Voltage: 3.3V-5V
- LED Type: 5mm Common-Cathode RGB
- Current: ~20-40 mA
- Control: Built-in ASIC, preprogrammed
- Type: Automatic

## 2. LM393 Speed Sensor
- Model: LM393 with perforated disc
- Groove Width: 5mm
- Voltage: 3.3V-5V
- Output: Digital
- Mounting: Fixed bolt hole
- Dimensions: 32 x 14mm

## 3. KY-012 Active Piezo Buzzer
- Voltage: 3.3V-5V
- Max Current: 30 mA
- Frequency: 2.5kHz ± 300 Hz
- Sound Output: ≥85 dB
- Temperature: -20°C to 70°C

## 4. KY-036 Metal-Touch Sensor
- Type: Analog/Digital
- Output: Both analog and digital
- Features: Adjustable threshold
- Detection: Conductivity change

## 5. KY-010 Light Barrier
- Voltage: 3.3V-5V
- Dimensions: 18.5 x 15mm (0.728" x 0.59")
- Output: Digital (HIGH when blocked)
- Type: Phototransistor + IR LED

## 6. KY-009 RGB LED Module
- Forward Voltage (Red): 1.8V
- Forward Voltage (Green/Blue): 2.8V
- Current: 20 mA
- Type: Common Cathode SMD

## 7. KY-029 2-Color LED
- Forward Voltage: 2.0-2.5V
- Current: 20 mA
- Colors: Red and Green
- Type: Common Cathode
